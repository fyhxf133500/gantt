# Changelog

记录项目的重要变更。

## [2026-04-26]
### Added
- 增加专业 CPM（Critical Path Method）关键路径计算，基于 Forward Pass / Backward Pass / Total Slack 识别关键任务。
- CPM 支持 FS、SS、FF 依赖类型与可选 lag。
- 关键路径任务自动标记 `isCritical`，普通任务和里程碑参与计算，父任务不参与 CPM 节点计算。
- 甘特图新增“关键路径”显示开关。
- 关键路径任务条和关键路径依赖箭头使用醒目红色样式展示。
- 检测到依赖循环时显示修复提示，并停止关键路径计算。
- 新增父任务内局部关键路径：点击父任务后，仅在该父任务的递归子任务范围内运行 CPM。
- 局部关键路径任务与依赖箭头使用橙色/虚线样式展示，且全局关键路径红色样式优先。
- 里程碑新增覆盖层视觉：以 16px 菱形和右侧任务名标签展示，支持普通、全局关键和局部关键状态。

### Changed
- 父任务关键状态由子任务派生显示，不参与 CPM 网络计算。
- CPM Forward Pass 使用当前甘特图任务开始日期作为最早开始下界，避免多个独立任务组被错误平移到同一起点。
- 依赖未显式设置 lag 时，CPM 不再从当前排期自动推导等待时间；任务间可见空档默认视为 slack。
- 点击页面空白区域会取消父任务选中状态，并隐藏局部关键路径。
- 局部关键路径任务样式从橙色填充改为橙色虚线边框，任务条本体保持原有颜色。
- 同时命中全局与局部关键路径时，保留红色全局填充并叠加橙色虚线局部标识；依赖线同理叠加偏移虚线。
- 缩小依赖箭头 marker 与线宽，降低箭头对任务条的视觉抢占。
- 里程碑局部关键状态改为橙色虚线外框，全局关键状态保持红色菱形。
- 提升甘特图任务浮窗层级，避免依赖箭头和里程碑覆盖层遮挡浮窗内容。
- 修复里程碑结束时间在任务列表中被第三方甘特库重置为开始时间的问题，列表和浮窗改用项目原始任务数据展示日期。
- 里程碑支持按起止时间显示可变长度长菱形，结束时间大于开始时间时不再只显示为一个点。
- 修正长菱形覆盖层与底层可拖拽时间条不重合的问题，让菱形左右边界跟随真实拖拽边界。
- 关键路径计算中的里程碑工期改为使用实际起止时间，保持与可变长度里程碑显示一致。
- localStorage 保存时会剥离依赖关系中的派生关键路径标记，保持旧数据兼容与存储干净。

## [2026-04-25]
### Added
- 依赖关系数据结构升级，支持 FS、SS、FF 三种依赖类型。
- 任务编辑弹窗支持添加、删除和编辑多条依赖关系。
- 新增依赖冲突确认弹窗，用户可选择自动调整后续任务或取消本次修改。
- 甘特图支持按依赖类型绘制直角依赖箭头。

### Changed
- 依赖校验从默认自动排程改为用户选择模式。
- 任务依赖旧数据兼容，从 `string[]` 自动迁移为 FS 类型依赖。
- 固定 Vite 开发端口为 `5173`，避免重启后端口漂移。

### Fixed
- 修复依赖箭头在横向滚动时超出甘特图可视区域的问题。
- 修复第三方甘特图多层滚动容器下，覆盖层误绑定完整 SVG 画布导致箭头左右溢出的问题。
- 记录本轮浏览器工具拒绝访问与滚动视口定位相关经验教训。

## [2026-03-29]
### Added
- 任务列表支持拖拽调整层级（拖到任务上成为子任务、拖到空白区变为顶级）。
- 拖拽目标高亮与拖拽态样式提示。
- 父任务自动汇总（开始/结束时间与进度自动计算）。
- 父任务删除时的选项弹窗（删除子任务或提升为顶级）。

### Changed
- 父任务在甘特图中使用独特样式并禁用拖拽。
- 父任务在编辑弹窗中禁用起止时间与进度编辑。

### Removed
- 子任务超出父任务时的处理弹窗。

## [2026-03-22]
### Added

- 层级防循环校验（禁止设置为自身或子孙节点）。
