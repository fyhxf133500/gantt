# Changelog

记录项目的重要变更。

## [2026-04-25]
### Added
- 依赖关系数据结构升级，支持 FS、SS、FF 三种依赖类型。
- 任务编辑弹窗支持添加、删除和编辑多条依赖关系。
- 新增依赖冲突确认弹窗，用户可选择自动调整后续任务或取消本次修改。
- 甘特图支持按依赖类型绘制直角依赖箭头。

### Changed
- 依赖校验从默认自动排程改为用户选择模式。
- 任务依赖旧数据兼容，从 `string[]` 自动迁移为 FS 类型依赖。
- 固定 Vite 开发端口为 `5173`，避免重启后端口漂移。

### Fixed
- 修复依赖箭头在横向滚动时超出甘特图可视区域的问题。
- 修复第三方甘特图多层滚动容器下，覆盖层误绑定完整 SVG 画布导致箭头左右溢出的问题。
- 记录本轮浏览器工具拒绝访问与滚动视口定位相关经验教训。

## [2026-03-29]
### Added
- 任务列表支持拖拽调整层级（拖到任务上成为子任务、拖到空白区变为顶级）。
- 拖拽目标高亮与拖拽态样式提示。
- 父任务自动汇总（开始/结束时间与进度自动计算）。
- 父任务删除时的选项弹窗（删除子任务或提升为顶级）。

### Changed
- 父任务在甘特图中使用独特样式并禁用拖拽。
- 父任务在编辑弹窗中禁用起止时间与进度编辑。

### Removed
- 子任务超出父任务时的处理弹窗。

## [2026-03-22]
### Added
- 任务编辑弹窗新增父任务选择与任务类型（普通/里程碑）。
- 任务层级结构支持展开/折叠，并在列表中展示缩进。
- 层级防循环校验（禁止设置为自身或子孙节点）。

### Changed
- useTasks 生成任务树并输出可视化列表（visibleTasks）。
- mockTasks 增加父子任务、依赖与里程碑示例。
- 本地存储持久化任务层级字段与展开状态。

## [2026-03-15]
### Added
- 任务新增/编辑/删除能力与 TaskFormModal 弹窗。
- 任务列表操作列与 新建任务按钮。
- localStorage 持久化（含初始化与自动保存）。
- 拖动甘特任务条更新任务时间（onDateChange）。
- 悬浮提示中文化与日期显示格式优化。

### Changed
- 甘特图视图与列表组件结构完善（GanttChart/TaskList）。
- 修复并统一 UTF-8 编码处理。

## [2026-03-14]
### Added
- 项目初始化（Vite + React + TypeScript）。
